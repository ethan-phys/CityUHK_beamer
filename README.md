# CityUHK Beamer Template

This repository contains a CityUHK-inspired `beamer` theme and a fully editable example deck. The example content is intentionally fictional: it uses a random "lantern dynamics" story so you can see the layout without mixing it with real talk material.

## Files

- `beamerthemeCityUHK.sty`: theme file with colors, backgrounds, title page, section pages, footline navigation, and closing slide helper.
- `cityu_beamer_example.tex`: example presentation using random placeholder content.
- `assets/cityuhk_closing.jpg`: background used by the title page and closing page.
- `assets/cityuhk_content.jpg`: background used by normal content pages.
- `assets/cityuhk_section.jpg`: background used by automatic section title pages.
- `assets/cityuhk_title.jpg`: extra title-style asset kept for experimentation.

## Compile

Run `pdflatex` twice so navigation, the table of contents, and page totals settle correctly:

```bash
pdflatex -interaction=nonstopmode cityu_beamer_example.tex
pdflatex -interaction=nonstopmode cityu_beamer_example.tex
```

The output is `cityu_beamer_example.pdf`.

## Editing Metadata

Edit these commands near the top of `cityu_beamer_example.tex`:

```tex
\title[Short Title]{Full Presentation Title}
\subtitle{Optional subtitle}
\author[Short Author]{Author One \and Author Two}
\institute[Short Institute]{Full institute name}
\date[Short Date]{Full date}
\conference{Conference or event name}
\titlenote{Optional note below the date}
```

## Background Mapping

The current theme uses these background macros:

```tex
\CityUHKClosingBg   % title page and closing page
\CityUHKContentBg   % normal content pages
\CityUHKSectionBg   % automatic section divider pages
```

The title page explicitly uses the closing background in the first frame:

```tex
\usebackgroundtemplate{\CityUHKFullBackground{\CityUHKClosingBg}}
```

## Position Controls

The title page text is positioned in `beamerthemeCityUHK.sty` using fixed TikZ coordinates. This avoids relying on fragile `\vspace` placement.

Conference label:

```tex
] at ([xshift=1.55cm,yshift=-0.4cm]current page.north west)
```

Title block:

```tex
] at ([xshift=4.0cm,yshift=-2.25cm]current page.north west)
```

Increase `xshift` to move text right. Make `yshift` more negative to move text down.

## Slide Types Included

- Title page with conference, title, authors, institute, date, and optional note.
- Outline slide generated by `\tableofcontents`.
- Automatic section divider slides generated by `\section{...}`.
- Regular content slides with bottom navigation.
- Example figure slide with explanation.
- Example pure math slide.
- Example derivation slide.
- Closing slide using `\CityUHKClosingFrame{...}{...}`.

## Reusing the Template

1. Copy `cityu_beamer_example.tex` to a new `.tex` file.
2. Replace the metadata and fictional content.
3. Keep `\section{...}` commands to preserve automatic section pages and navigation.
4. Replace or adjust the background images in `assets/` if needed.
5. Compile twice with `pdflatex`.
